# Ethics Integration Prompts (Simple)

## Prompt 1: Risk Assessment
```
AI system: [describe what AI does and how]
Users: [who uses it and for what]

Identify main risks:
1. Could this be unfair to some groups?
2. Are there privacy concerns with the data?
3. Do users need to understand how AI works?
4. What happens if the AI makes mistakes?

Suggest simple ways to address each risk.
```

## Prompt 2: Trust Building
```
AI product: [describe your solution]
User context: [how technical your users are]

How can we build user trust:
1. What should users understand about the AI?
2. What control should they have?
3. How should we handle errors?
4. What would make them comfortable using this?

Focus on practical features for MVP.
```

## Prompt 3: Ethical Business Model
```
Business model: [summarize your canvas]
Main ethical concerns: [list key risks]

How can we integrate ethics:
1. Add ethical value to our main benefit
2. Include ethical safeguards in key activities
3. Budget for responsible AI practices
4. Make ethics a competitive advantage

Keep suggestions realistic for student team.
```
