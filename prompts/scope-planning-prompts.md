# Scope Planning Prompts (Simple)

## Prompt 1: Feasibility Check
```
Team skills: [list what team can do]
Want to build: [describe AI product concept]
Timeline: 3 weeks

Honest assessment:
1. Can this team build this in 3 weeks?
2. What's most likely to go wrong?
3. What should we simplify?
4. What backup approach would work?

Be realistic about student skill level.
```

## Prompt 2: MVP Features
```
Product concept: [what you want to build]
User workflow: [how someone would use it]

For 3-week MVP, what features:
1. Must have (or product doesn't work)
2. Should have (important for good experience)
3. Could have (nice if time permits)
4. Won't have (too complex)

Focus on demonstrating core AI value.
```

## Prompt 3: Weekly Planning
```
MVP scope: [what you'll build in 3 weeks]
Team capacity: [team size and hours/day]

Create weekly plan:
Week 1: [what to complete - focus on core AI]
Week 2: [what to complete - focus on user experience]
Week 3: [what to complete - focus on demo prep]

What could go wrong each week and how to handle it?
```

## Prompt 4: Success Metrics
```
AI product: [describe your 3-week deliverable]
Target users: [who will test it]

Define simple success criteria:
1. How will users measure if this works?
2. How will we know the AI is performing well?
3. What evidence shows this could be valuable?
4. What should we demo to impress stakeholders?

Keep metrics simple and measurable.
```
