# Business Canvas Prompts (Simple)

## Prompt 1: Customer & Problem
```
AI product idea: [describe your concept]

Help identify:
1. Who would actually use this? (be specific)
2. What problem does it solve for them?
3. How do they solve this problem now?
4. Why would they switch to an AI solution?

Keep answers practical and specific.
```

## Prompt 2: Value & Revenue
```
Product: [describe solution]
Users: [who uses it]

Help determine:
1. What's the main value this creates?
2. How would users measure success?
3. How could this make money? (internal value or external revenue)
4. What would be realistic pricing/value?

Focus on simple, testable models.
```

## Prompt 3: Resources & Costs
```
AI product: [describe what you're building]
Technical approach: [your AI method]

What do we need:
1. Essential data sources
2. Key technology/tools
3. Critical skills/expertise
4. Rough cost estimate for 3 months

Keep it simple and focused on necessities.
```
