# Team Formation Prompts (Simple)

## Prompt 1: Project-Team Match
```
I have these project ideas: [list 3-4 ideas]
These people with skills: [list 4-6 people and skills]

For a 3-week AI project, suggest the best project-team combinations.
Consider: skill match, realistic scope, team chemistry.

For each suggestion, explain why it would work and what the main risks are.
```

## Prompt 2: Role Assignment
```
Our team: [describe team members and skills]
Our project: [describe what you're building]
Timeline: 3 weeks

Suggest who should do what:
- Technical lead (AI development)
- Product lead (user research, requirements)
- Integration lead (user experience, testing)

Consider each person's strengths and learning goals.
```

## Prompt 3: Realistic Scope Check
```
Team: [X] people with skills: [list skills]
Want to build: [describe project]
Timeline: 3 weeks

Be honest: Is this realistic?
If not, what should we simplify or change?
What backup approaches should we consider?
```
